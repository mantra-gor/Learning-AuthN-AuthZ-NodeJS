API Documentation

POST /api/v1/register

Register a new user.

        Request:
        {
          "name": "mantra gor", 
          "email": "johndoe@gmail.com",
          "role": "Student"
          "password": "11223344", 
        }
        
        Response:
        {
          "success": true,
          "message": "Successfully Created an Account"
        }

POST /api/v1/student

Protected route for students. Requires authentication and student role.

        Headers:
        {
          "Authorization": "Bearer YOUR_ACCESS_TOKEN"
        }
        
        Response:
        {
            "success": true,
            "message": "Welcome To Student"
        }

POST /api/v1/admin

Protected route for admins. Requires authentication and admin role.

        Headers:
        {
          "Authorization": "Bearer YOUR_ACCESS_TOKEN"
        }

        Response:
        {
            "success": true,
            "message": "Welcome To Admin"
        }
